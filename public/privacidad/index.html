Política de privacidad

En UnaHojaTools respetamos tu privacidad y nos comprometemos a proteger tus datos personales conforme al Reglamento (UE) 2016/679 (RGPD) y la normativa española vigente.

1. Responsable del tratamiento

Titular: UnaHojaTools
Sitio web: https://unahojatools.com
Correo electrónico de contacto: contacto@unahojatools.com

Este sitio es un proyecto informativo que ofrece herramientas de cálculo orientativas.

2. Finalidad del tratamiento

Los datos personales que puedan recabarse a través del sitio web se utilizan con las siguientes finalidades:

  • Responder consultas enviadas mediante el formulario de contacto.
  • Garantizar el correcto funcionamiento técnico del sitio.
  • Mostrar publicidad contextual o personalizada mediante Google AdSense, si el usuario lo acepta.

3. Datos que se recogen

Este sitio web no requiere registro ni creación de cuenta.

Las herramientas de cálculo funcionan íntegramente en el navegador del usuario y no envían los datos introducidos a nuestros servidores.

Solo podrán tratarse datos identificativos básicos (como correo electrónico) si el usuario los facilita voluntariamente a través del formulario de contacto.

4. Base jurídica

  • Consentimiento del interesado (art. 6.1.a RGPD).
  • Interés legítimo en garantizar el funcionamiento y seguridad del sitio.

5. Publicidad y terceros

Este sitio puede mostrar anuncios gestionados por Google AdSense.

Google puede utilizar cookies y tecnologías similares para mostrar anuncios relevantes. El usuario puede aceptar o rechazar la carga de anuncios desde el panel de consentimiento.

Más información en: Políticas de publicidad de Google .

6. Conservación de datos

Los datos enviados mediante contacto se conservarán únicamente durante el tiempo necesario para responder a la consulta.

7. Derechos del usuario

El usuario puede ejercer sus derechos de:

  • Acceso
  • Rectificación
  • Supresión
  • Limitación
  • Oposición
  • Portabilidad

Para ello puede escribir a: contacto@unahojatools.com

8. Seguridad

Se aplican medidas técnicas razonables para proteger la información tratada.

9. Modificaciones

Esta política puede actualizarse para adaptarse a cambios normativos o técnicos.
